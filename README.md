# 👩🏻‍💻 Aila Bentes

Meu nome é Aila Bentes Moraes e este projeto foi desenvolvido na Universidade Federal da Fronteira Sul (UFFS) em conjunto com mais duas integrantes da equipe.
O projeto se chama Animal Love e foi criado com o objetivo de contribuir para a proteção e o bem-estar animal por meio de uma plataforma digital acessível e intuitiva.
A ideia surgiu da necessidade de facilitar a conexão entre pessoas que desejam ajudar animais e instituições que atuam nessa causa. Por meio do sistema, os usuários podem
adotar animais, cadastrar pets para adoção, realizar denúncias de maus-tratos, cadastrar ONGs, participar como voluntários e fazer doações para apoiar ações de proteção animal.
Para o desenvolvimento do projeto foram utilizadas tecnologias de front-end, como HTML5, CSS3 e Tailwind CSS, e algumas linhas de JavaScript permitindo a criação de uma interface moderna, organizada e 
responsiva para diferentes dispositivos.
O Animal Love busca utilizar a tecnologia como ferramenta de impacto social, promovendo a adoção responsável, o combate aos maus-tratos e o fortalecimento da rede de apoio aos animais.
